# Problem 7

10001st prime

## Description
By listing the first six prime numbers: 2, 3, 5, 7, 11, and 13, we can see that the 6th prime is 13.

What is the 10 001st prime number?

## Compilation
### Option 1
`$ make`
### Option 2
1. `$ mkdir -p bin`

2. `$ gcc -Wall -O3 -o bin/problem7 main.c`

## Execution
`$ ./bin/problem7`
