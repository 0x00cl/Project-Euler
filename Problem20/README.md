# Problem 20

Factorial digit sum

## Description

n! means n × (n − 1) × ... × 3 × 2 × 1

For example, 10! = 10 × 9 × ... × 3 × 2 × 1 = 3628800,
and the sum of the digits in the number 10! is 3 + 6 + 2 + 8 + 8 + 0 + 0 = 27.

Find the sum of the digits in the number 100!

## Compilation
### Option 1
`$ make`
### Option 2
1. `$ mkdir -p bin`

2. `$ gcc -Wall -Wextra -O3 -o bin/problem20 main.c`

## Execution
`$ ./bin/problem20`
