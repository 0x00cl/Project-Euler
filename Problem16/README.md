# Problem 16

Power digit sum

## Description
2^15 = 32768 and the sum of its digits is 3 + 2 + 7 + 6 + 8 = 26.

What is the sum of the digits of the number 2^1000?

## Compilation
### Option 1
`$ make`
### Option 2
1. `$ mkdir -p bin`

2. `$ gcc -Wall -O3 -o bin/problem16 main.c`

## Execution
`$ ./bin/problem16`
