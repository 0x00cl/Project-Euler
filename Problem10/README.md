# Problem 10

Summation of primes

## Description
The sum of the primes below 10 is 2 + 3 + 5 + 7 = 17.

Find the sum of all the primes below two million.

## Compilation
### Option 1
`$ make`
### Option 2
1. `$ mkdir -p bin`

2. `$ gcc -Wall -O3 -o bin/problem10 main.c`

## Execution
`$ ./bin/problem10`
